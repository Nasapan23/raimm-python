# Programa Scurta - Course 4 (Computer Vision OpenCV)

## 1) Obiectivul lectiei (5 min)
- Ce inseamna computer vision in robotica.
- Pipeline rapid: imagine -> preprocesare -> detectie -> decizie.

## 2) BW Transform (10 min)
- Conversie imagine color in grayscale.
- Threshold pentru imagine alb-negru.
- Demo: `exemplul_1_bw_transform.py`

## 3) Canny Edge Detection (10 min)
- Blur pentru reducerea zgomotului.
- Praguri Canny si detectie muchii.
- Demo: `exemplul_2_canny.py`

## 4) Contours (10 min)
- Masca binara si extragere contururi.
- Bounding boxes pe obiecte detectate.
- Demo: `exemplul_3_contours.py`

## 5) Face Recognition Basic (15 min)
- Detectie fata cu Haar Cascade.
- Enrolare persoane cunoscute.
- Comparare simpla si etichetare.
- Demo: `exemplul_4_face_recognition.py`

## 6) Bonus Exercise - YOLO Live + Tkinter (15 min)
- Detectie automata de obiecte pe camera live cu YOLOv8.
- UI in Tkinter (Start/Stop + lista obiecte detectate in timp real).
- Demo: `bonus_exercise_yolo_live_tkinter.py`

## 7) Inchidere / Tema scurta (5 min)
- Tema: schimba pragurile si observa diferentele.
- Bonus extins: ajusteaza pragul de incredere si urmareste impactul pe detectii.
